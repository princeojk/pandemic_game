# Quarantine Quest 🎮

This is a decision-tree driven pandemic simulation game built with Flutter Web.

Players navigate a branching decision tree, making critical policy choices that dynamically 
affect two nations: the **Green Country** and the **Red Country**  across key metrics.

## 🛠️ Technologies Used

- **Flutter**: For building a sleek and responsive user interface with Material Design.
- **Hive**: Fast and offline-friendly data storage solution.
- **Cloud Firestore**: Real-time database for syncing game data seamlessly.
